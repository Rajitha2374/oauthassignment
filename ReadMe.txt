This project includes basic implementation of authentication for Github using OAuth2.0.

Getting Started

These instructions will explain the direction to deploy the application for testing.

Prerequisites

You have to install nodeJS and relevant dependencies.
	   npm install

Installing

1. Clone the repository to your machine and open the terminal in that folder.
2. Run the following command in the terminal to run the app.
        node app.js
3. Open the browser and go to http://localhost:4000
4. Click on the login button and it will redirect you to welcome page
